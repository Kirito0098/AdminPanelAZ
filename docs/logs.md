# Журналы

## Зачем эта страница

Здесь смотрят **логи и историю событий**: кто подключался, что происходило с OpenVPN, какие действия выполняли администраторы, кто скачивал QR-коды.

## Кто может пользоваться

Все роли (набор вкладок зависит от включённых модулей и роли).

## Как открыть

Боковое меню → **Журналы**.

Данные собираются с **активного узла** — переключите узел в шапке, если нужен другой сервер.

---

## Вкладки

| Вкладка | Что показывает |
|---------|----------------|
| **Подключения** | Журнал подключений и отключений клиентов |
| **OpenVPN события** | Технические события OpenVPN (ошибки, переподключения) |
| **Действия** | Аудит: кто из администраторов что сделал в панели |
| **QR-скачивания** | Когда и кем скачивали конфиги по QR |
| **OVPN сокеты** | Активные сетевые соединения OpenVPN |

## Пошаговые сценарии

### Найти, когда клиент подключался

1. Откройте **Журналы** → вкладка **Подключения**
2. Введите имя клиента в **Поиск**
3. Листайте страницы кнопками **Назад** / **Вперёд**

### Посмотреть ошибки OpenVPN

1. Вкладка **OpenVPN события**
2. В фильтре **Уровень** выберите **ERROR** или **WARN**
3. При необходимости выберите конкретный профиль OpenVPN

### Узнать, кто удалил клиента

1. Вкладка **Действия**
2. Поиск по ключевому слову (имя клиента, «delete», «удал»)
3. Экспорт журнала — кнопка **Скачать** (если доступна)

### Сохранить лог на компьютер

1. Откройте нужную вкладку
2. Нажмите **Скачать** — файл сохранится на ваш компьютер
3. Или **Копировать** — текст попадёт в буфер обмена

## Кнопки и фильтры

| Элемент | Что делает |
|---------|------------|
| **Поиск** | Фильтр строк по тексту |
| **Уровень** | ERROR / WARN / INFO / все |
| **Хвост** | Автоматически прокручивать к новым строкам |
| **Номера строк** | Показать номера строк в логе |
| **Обновить** | Загрузить свежие данные |
| **Копировать** | Скопировать видимые строки |
| **Скачать** | Сохранить лог в файл |

Страница может обновляться автоматически — следите за меткой «обновлено» в шапке.

## Частые вопросы

**Логи пустые, узел офлайн**  
Проверьте связь с VPN-сервером на странице **Узлы**. Пока сервер недоступен, новые записи не поступают.

**Чем «Подключения» отличается от NOC?**  
NOC — картина «сейчас». Журналы — **история** за прошедшее время с возможностью поиска и скачивания.

**Кто видит вкладку «Действия»?**  
Зависит от настроек модулей. Обычно доступна администраторам и наблюдателям.

---

## Будьте осторожны

- Журналы могут содержать IP-адреса клиентов — не публикуйте их
- Экспорт больших логов может занять время
- Старые записи со временем удаляются (зависит от настроек сервера)
