# AZ-WARP

## Зачем эта страница

**AZ-WARP** направляет трафик **отдельных сайтов и сервисов** через Cloudflare WARP — это другой способ обхода блокировок, точечный, для конкретных доменов (ChatGPT, Gemini и др.).

## Кто может пользоваться

Только **администратор**.

## Как открыть

Боковое меню → **AZ-WARP**.

Если модуль не установлен — страница предложит установку.

---

## Вкладки

| Вкладка | Для чего |
|---------|----------|
| **Домены** | Список сайтов, которые идут через WARP |
| **IP-подсети** | Диапазоны IP для WARP-маршрутизации |
| **Мониторинг** | Статус, трафик, логи WARP |
| **Настройки** | Параметры работы AZ-WARP |

---

## Пошаговые сценарии

### Включить AZ-WARP

1. Откройте **AZ-WARP**
2. Если видите предложение установить — следуйте инструкции на экране
3. Включите переключатель в верхней части страницы

### Добавить сайт для WARP

1. Перейдите на вкладку **Домены**
2. Нажмите добавить домен или импортируйте список
3. Есть готовые списки для Gemini, ChatGPT — можно включить одной кнопкой
4. Нажмите **Синхронизировать**, если нужно применить на сервер

### Проверить, работает ли WARP

1. Вкладка **Мониторинг** — статус службы и трафик
2. Вкладка с логами — ошибки подключения, если что-то не так

### Диагностика проблем

В разделе **Doctor** (если есть на странице) — автоматическая проверка типичных проблем с WARP.

## Кнопки (вкладка Домены)

| Кнопка | Что делает | Когда использовать |
|--------|------------|-------------------|
| **Добавить домен** | Один сайт в список WARP | Для точечного добавления |
| **Импорт** | Загрузка списка доменов | Много сайтов сразу |
| **Встроенные списки** | Gemini, ChatGPT и др. | Быстрый старт |
| **Синхронизировать** | Применить список на сервер | После изменений |
| **Удалить** | Убрать домен из WARP | Сайт больше не нужен через WARP |

## Частые вопросы

**Чем WARP отличается от обычной маршрутизации?**  
Обычная маршрутизация (CIDR) работает по спискам провайдеров. WARP — отдельный канал через Cloudflare для выбранных доменов.

**Нужен ли WARP всем?**  
Нет, это опциональный модуль. Включайте, если нужен доступ к конкретным зарубежным сервисам.

**WARP и Конфиг AntiZapret — связаны?**  
Да, в **Конфиг AntiZapret** есть переключатели WARP. AZ-WARP — детальное управление доменами и мониторинг.

---

## Будьте осторожны

- Отключение WARP может нарушить доступ к сайтам, которые шли через него
- Массовый импорт доменов — проверьте список перед синхронизацией
- Установка и обновление WARP требуют времени — не прерывайте процесс
