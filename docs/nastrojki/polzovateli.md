# Пользователи

## Зачем этот раздел

Здесь создают **учётные записи** для входа в панель и назначают **роли**: кто может только смотреть, кто управляет своими конфигами, кто полный администратор.

## Кто может пользоваться

Только **администратор**.

## Как открыть

**Настройки** → **Пользователи**.

---

## Роли простыми словами

| Роль | Что может |
|------|-----------|
| **Администратор** | Всё: клиенты, узлы, настройки, бэкапы |
| **Пользователь** | Свои VPN-конфиги, скачивание, QR |
| **Наблюдатель** | Только просмотр: мониторинг, журналы, конфиги без изменений |

---

## Пошаговые сценарии

### Создать пользователя

1. Нажмите **Добавить пользователя**
2. Укажите **логин** и **пароль**
3. Выберите **роль**
4. При необходимости укажите **Telegram ID** — для входа через Telegram и получения конфигов в боте
5. Сохраните

### Привязать Telegram к пользователю

1. Узнайте Telegram ID пользователя (через бота @userinfobot или команду `/link` в вашем боте)
2. В карточке пользователя укажите **Telegram ID**
3. Сохраните

Подробнее о Telegram: [Telegram.md](../Telegram.md).

### Удалить пользователя

1. Найдите пользователя в списке
2. Нажмите **Удалить**
3. Подтвердите — учётная запись исчезнет, VPN-конфиги на сервере **не удаляются** автоматически

### Дать пользователю доступ только к своим конфигам

1. Создайте пользователя с ролью **Пользователь**
2. При создании клиента на **Конфигурации** укажите этого пользователя **владельцем**

## Частые вопросы

**Сколько администраторов можно?**  
Сколько нужно, но для безопасности лучше минимум — один основной и запасной.

**Наблюдатель видит чужие конфиги?**  
Обычно видит список для мониторинга, но не может их менять или скачивать — зависит от настроек.

**Можно ли сменить роль?**  
Да — отредактируйте пользователя и выберите другую роль.

---

## Будьте осторожны

- Не давайте роль **Администратор** без необходимости
- Удаление пользователя **не отключает** его VPN на сервере — при необходимости заблокируйте клиента отдельно
- Передавайте начальный пароль по защищённому каналу, не в открытом чате
