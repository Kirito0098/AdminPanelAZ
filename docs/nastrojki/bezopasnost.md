# Доступ к панели (безопасность)

## Зачем этот раздел

Защита **самой панели** от посторонних: ограничение по IP, блокировка подозрительных попыток входа, смена секретных ключей.

## Кто может пользоваться

Только **администратор**.

## Как открыть

**Настройки** → **Доступ к панели**.

---

## Пошаговые сценарии

### Ограничить вход только с ваших IP

1. Найдите блок **Белый список IP**
2. Добавьте ваши IP-адреса (домашний, офисный, VPN админа)
3. Включите whitelist
4. Сохраните

> Узнайте свой IP заранее (например, через сайт «мой IP»). Иначе можете **заблокировать себя**.

### Посмотреть заблокированных сканеров

1. Блок **Активные баны** — IP, которые панель временно заблокировала за подозрительную активность
2. При необходимости снимите бан вручную

### Посмотреть активные сессии

1. Блок **Активные сессии** — кто сейчас залогинен в панель
2. Можно завершить чужую сессию при подозрении на взлом

### Сменить секретные ключи (ротация)

1. Откройте **Мастер ротации секретов**
2. Выберите, что менять: ключ сессий, API-ключи, токен Telegram
3. Пройдите шаги с подтверждением
4. После смены ключа сессий **все пользователи** должны войти заново

---

## Частые вопросы

**Включил whitelist и не могу войти**  
Заходите с IP, который добавили в список. Если заблокировали себя — правьте настройки через SSH на сервере или временно отключите whitelist там же.

**Откуда берутся баны?**  
Панель автоматически блокирует IP после множества неудачных попыток входа или сканирования.

**Когда менять секреты?**  
При подозрении на утечку, после увольнения администратора, раз в несколько месяцев для профилактики.

---

## Будьте осторожны

- **Whitelist** без запасного IP — риск потерять доступ к панели
- **Ротация SECRET_KEY** — все выйдут из системы, запланируйте время
- **Ротация Telegram token** — бот перестанет работать, пока не обновите токен в BotFather и в панели
