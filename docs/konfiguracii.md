# Конфигурации (VPN-клиенты)

## Зачем эта страница

Здесь вы видите всех VPN-клиентов и управляете ими: создаёте, скачиваете конфиг, показываете QR-код, блокируете, ставите лимит трафика.

## Кто может пользоваться

| Действие | Администратор | Пользователь |
|----------|:-------------:|:------------:|
| Просмотр списка | ✓ | ✓ (свои + доп. доступ) |
| Скачивание, QR | ✓ | ✓ (свои + доп. доступ) |
| Создание | ✓ | ✓ (если разрешил админ, в рамках квоты) |
| Блокировка, удаление | ✓ | ✓ только свои |
| Синхронизация, импорт/экспорт CSV | ✓ | — |

## Как открыть

Боковое меню → **Конфигурации**.

**Важно:** в шапке выбран **активный узел**. Все клиенты и действия относятся только к этому VPN-серверу.

---

## Пошаговые сценарии

### Создать нового клиента

1. Убедитесь, что выбран нужный узел в шапке
2. Нажмите **Новый клиент**
3. Заполните форму:
   - **Имя** — латиница, цифры, `_` и `-`, до 32 символов
   - **Тип VPN** — OpenVPN или WireGuard/AmneziaWG
   - **Срок сертификата** — сколько дней действует (для OpenVPN)
   - **Описание** — по желанию
4. Нажмите **Создать**

### Скачать конфиг или показать QR

1. Найдите карточку клиента в списке
2. Нажмите **Скачать** — получите файл для приложения VPN
3. Или **QR-код** — отсканируйте телефоном для быстрого подключения

### Подтянуть клиентов с сервера (синхронизация)

Если клиенты уже были созданы на VPN-сервере, но не видны в панели:

1. Выберите нужный узел
2. Нажмите **Синхронизировать**
3. Подождите завершения — список обновится

### Действия с клиентом (расширенное меню)

Нажмите на карточку клиента или кнопку действий — откроется окно с разделами:

- **Владелец** — кому принадлежит конфиг
- **Управление** — блокировка, продление, лимит трафика, отключение сессии
- **Файлы и доступ** — скачать, QR, одноразовая ссылка
- **Теги** — метки для группировки и фильтрации
- **Опасные действия** — постоянная блокировка, удаление профиля

## Кнопки в шапке страницы

| Кнопка | Что делает | Когда использовать |
|--------|------------|-------------------|
| **Синхронизировать** | Загружает клиентов с диска VPN-сервера | После добавления узла или ручного создания на сервере |
| **Экспорт CSV** | Сохраняет список клиентов в файл | Для резервной копии или переноса |
| **Импорт CSV** | Загружает клиентов из файла | Массовое добавление; опционально лимит трафика и блок (`traffic_limit_bytes`, `traffic_limit_days`, `block_mode`) |
| **Новый клиент** | Создаёт VPN-профиль | Когда нужен новый пользователь |

## Фильтры и поиск

- **Поиск** — по имени клиента
- **Фильтры** — онлайн, заблокированные, с истёкшим сроком
- **Теги** — показать только клиентов с выбранной меткой
- **Массовый выбор** — отметить несколько карточек и выполнить действие сразу (блокировка, продление, удаление)

## Частые вопросы

**Почему клиент не появился после создания на сервере?**  
Нажмите **Синхронизировать** — панель подтянет данные с диска.

**Чем отличается временная и постоянная блокировка?**  
Временная — на заданное число дней, потом снимается. Постоянная — до ручного снятия блокировки.

**Что такое одноразовая ссылка?**  
Ссылка для скачивания конфига, которая работает ограниченное время — удобно отправить клиенту без файла.

**Лимит конфигураций исчерпан**  
Администратор может изменить лимит в настройках или удалить неиспользуемые профили.

## Импорт CSV — колонки

Обязательные: `client_name`, `vpn_type`. Опционально: `owner_username`, `cert_expire_days`, `description`, `traffic_limit_bytes`, `traffic_limit_days`, `block_mode`.

| Колонка | Значение |
|---------|----------|
| `traffic_limit_bytes` | Лимит в байтах (целое число) |
| `traffic_limit_days` | Период учёта: `1`, `7` или `30` |
| `block_mode` | `permanent`, `temp` (7 дней по умолчанию) или `temp:N` |

Пример файла: [examples/vpn-configs-import.example.csv](examples/vpn-configs-import.example.csv). В HA `sync_mode=auto` политики из CSV реплицируются на replica (как при применении шаблона).

## Будьте осторожны

- **Удалить профиль** — клиент потеряет доступ, восстановление только через новое создание
- **Блокировать навсегда** — клиент не сможет подключиться, пока вы не снимете блок
- **Импорт CSV** — может создать много клиентов сразу; проверьте файл перед загрузкой
- Все действия привязаны к **активному узлу** — не перепутайте сервер
